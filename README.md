# Sistema de Login Básico en Python

Sistema básico de autenticación desarrollado en Python como proyecto escolar para la asignatura de **Introducción a la Programación**.

El proyecto permite registrar usuarios, iniciar sesión y almacenar la información de manera local utilizando archivos JSON. Además, las contraseñas son protegidas mediante encriptación SHA-256.

## Características

- Registro de usuarios
- Inicio de sesión
- Encriptación de contraseñas
- Almacenamiento local con JSON
- Interfaz desarrollada con:
  - Streamlit
  - Flet
- Código modular y organizado

## Tecnologías utilizadas

- Python
- Streamlit
- Flet
- JSON
- SHA-256
- Git y GitHub

## Objetivo del proyecto

Este proyecto fue desarrollado como práctica académica con el objetivo de aprender:

- Modularización en Python
- Manejo de archivos
- Interfaces gráficas
- Validación de datos
- Organización de proyectos

## Ejecución del proyecto

### Ejecutar versión Streamlit

```bash
py -m streamlit run ui/streamlit_app.py
```

### Ejecutar versión Flet

```bash
py -m ui.flet_app
```

---

### Instalación de dependencias

Antes de ejecutar el proyecto, instale las siguientes librerías:

```bash
pip install streamlit
```

```bash
pip install flet
```
